## 游戏结构（4 关 + 2 段过场）

- **第一关**：三体信号监测员（扫雷找信号，10×15）

- **剧情过场**：面壁计划
- **第三关**：威慑纪元（判断真假威胁）
- **剧情过场**：掩体纪元

## 整体设计思路

- **视角切换**：三体视角（第1、2、4关）和地球视角（第3关）交替
- **玩法风格**：以"反应+判断"类玩法为主
- **剧情呈现**：大事件通过过场动画或文字交代，关键事件做成可玩关卡
- **难度递进**：从简单的扫雷、躲避，到复杂的判断、时机把握
- **重玩价值**：黑暗森林威慑度总评分（S/A/B/C）+ ETO 隐藏线索收集
- **难度分级**：三档难度（简单 / 普通 / 困难）
- **可访问性**：色觉障碍模式、单声道音频、字幕提示

## 第一关：信号监测员

### 场景设定

玩家是三体世界的信号监测员，负责监测来自地球的信号。

### 核心机制

- 表面是扫雷游戏，实际要靠声音找坐标
- **扫雷网格：10 行 × 15 列（共 150 格）**
- 离目标越近，噪音音调越接近"哔"（越远越接近"嗡"）
- **音调共分 5 档**：嗡 / 嗡- / 中 / 哔- / 哔（越近越尖锐）
- 既要防雷又要找信号，双重紧张感
- **三次容错机制**：踩雷不直接死亡，而是"暴露坐标 3 秒"——三体舰队获得逼近提示。三次后失败

### 操作规则

- 点击格子扫雷
- 通过声音判断目标位置
- **副反馈机制**（除声音外）：
  - 微震动（移动端 Vibration API）
  - 屏幕边缘光晕亮度（距离越近越亮）
- 找到信号源 = 成功
- 踩到雷（容错 3 次内）= 暴露坐标 3 秒
- 踩到雷（第 4 次）= 失败

### 失败条件

- 三次容错全部用完
- 倒计时归零（普通/困难模式）

### 视觉风格

- 扫雷格子用 SVG `<rect>` 阵列，深底 `#05060a`
- 未点击格子：边框 `#2a2f3a`
- 已点击格子：填充 `#0a0d14`
- 信号源位置：冷青 `#7fd4e8` 微光脉动
- 雷的位置（暴露坐标时）：警告红 `#d9455f` 闪烁 3 秒

## 第二关：飞船驾驶员

### 场景设定

玩家是三体飞船的驾驶员，在执行飞行任务。

### 核心机制

- 飞船在航行中
- 玩家初始在左下角，要躲避障碍，航行到右上角
- 图中有两个红点（外星文明），附近会不断出现移动红圈
- 红圈总覆盖范围能阻断左下→右上的航线，但会周期性移动形成"窗口期"
- 行星可借助引力加速但不能撞上

### 操作规则（明确方案）

- **控制方案**：鼠标点击目标点，飞船沿贝塞尔曲线移动（若经过行星引力范围则弧形绕行）
- **触屏**：单击即点击
- 躲避红点扫描圈
- 到右上角 = 成功
- 被扫描到 = 失败（容错 2 次后强制失败）

### 辅助反馈

- **扫描圈窗口期**：当红圈处于"可通过"窗口时，边缘亮起绿色 `#a8e8c8` 脉冲
- **引力等势线**：行星周围显示同心圆等势线，玩家能预判加速轨迹
- **轨迹预览**：鼠标悬停时显示贝塞尔路径（半透明虚线）

### 失败条件

- 飞船进入扫描圈 3 次（第 3 次失败）
- 撞上行星
- 撞上红点本体

### 视觉风格

- 星图背景用 `StarsField`（SVG 星点）
- 飞船：SVG 三角形图标 `#7fd4e8`
- 红点：圆形 `#d9455f`
- 扫描圈：径向渐变红，半透明
- 行星：圆形 `#e8d87f` + 引力等势线（同心圆）

## 第三关：威慑纪元

### 场景设定

玩家是罗辑，手指在威慑开关上。各种信息涌来，玩家要判断哪个是真正的威胁。

### 开始阶段

- 纪元开始时，屏幕上显示**三个判断标准**（左上、右上、左下）
- 每次游戏的判断标准都不同（随机生成）
- 每个判断标准用**文字 + 图形示例**显示
- 示例：
  - 左上："真的威胁：颜色不协调" + 示例图形（左边"不协调"，右边"协调"）
  - 右上："真的威胁：形状不规则" + 示例图形（左边"不规则"，右边"规则"）
  - 左下："真的威胁：动态混乱" + 示例图形（左边"混乱"，右边"规律"）

### 核心机制（v2.0 调整）

- 第一条信息的出现时间：**开始后，过 2-3 秒出现**（随机）
- 之后每条信息的出现时间间隔：**上一条出现后，过 2-3 秒再出现下一条**（随机）
- **每条信息持续 3-3.5 秒**（从 v1.0 的 2 秒延长，缓解认知压力）
- 信息在屏幕**上半部分（上 3/4）**的随机位置出现
- 信息**不会重叠**
- **总共显示 10-12 条信息**（从 v1.0 的 15 条降低）
- **真假比例 1:3**（10 条信息中 1 条真威胁 + 9 条假信息）
- 玩家要根据三个判断标准，判断哪条是真的威胁

### 操作

- 屏幕底部有一个按钮（威慑开关）
- 玩家看到真的威胁时，点击按钮
- 点击正确 = 威慑成功 → 进入掩体纪元剧情
- 点击错误 = 误判 → **扣 30% 进度**（从 v1.0 的直接失败改为代价递增）
- **第二次按错 = 游戏结束**

### 辅助反馈

- **真威胁出现前 0.5 秒**：屏幕四周有红光脉冲预警
- **判断标准可回顾**：按空格键暂停 1 次，显示三个标准（参考《低语》"可暂停"原则）
- **进度条**：底部显示"威慑值"进度条，随玩家表现增减

### 失败条件

- 第二次按错（即累计误判 2 次）
- 真威胁出现但未点击（持续到下一条出现）

### 视觉风格

- 信息卡片：SVG `<rect>` + `<text>`，半透明深色底 `#0a0d1480`
- 真威胁：边框 `#d9455f` 红色脉冲
- 假信息：边框 `#2a2f3a` 灰色
- 威慑按钮：底部居中，圆形 `#7fd4e8`
- 进度条：底部宽度 60%，颜色随威慑值变化（高：青；低：红）

## 第四关：逃离二向箔（v2.0 关键重构）

### 场景设定

飞船"星环号"出现一些小故障。玩家要手动校准逃逸参数，在完全二维化前启动逃逸程序。

### 核心机制（v2.0 用户确认版）

#### 4.1 一次性出所有环（**不分波次**）

- 所有激活环**同时显示**
- 玩家**自由切换鼠标目标**
- **自由选择激活顺序**（不强制顺序）
- 操作连续、无中断，保留"极限逃生"的紧迫感

#### 4.2 环数按难度分级

| 难度 | 环数 | 体验定位 |
|------|------|---------|
| **简单** | **3 个环** | 适合新手 / 休闲玩家 |
| **普通** | **4 个环** | 平衡体验（原版设计） |
| **困难** | **5 个环** | 极限挑战，要求高反应 + 高策略 |

#### 4.3 各环白点速度差异化

- 每个环有**独立转速**，差异 ≥ 30%
- 玩家必须"看菜吃饭"——先打慢的，还是先抢难的
- 二维化进度推进时，所有环转速按比例加速（基础速度 × (1 + 进度%)）

**转速建议区间**：
- 简单（3 环）：1.5-3.0 秒/圈
- 普通（4 环）：1.0-2.0 秒/圈
- 困难（5 环）：0.8-1.5 秒/圈

#### 4.4 判定窗口

- 每个激活环上画 **判定区域弧**（30° 绿色弧段 `#a8e8c8`）
- 玩家能"读"出最佳点击时机
- 白点转动到判定弧内点击 = 激活成功

#### 4.5 失败与抢救

- **失败条件**：二维化进度条满了（时间耗尽）
- **慢动作抢救**：第 3 个参数激活失败时，进入 3 秒慢动作，可抢救一次
- 第 4/5 个参数失败 = 直接失败

### 操作规则

1. 飞船固定在屏幕中央，**不能移动**
2. 左上角有进度条，显示"二维化进程"
3. 屏幕上有 3/4/5 个环（按难度），每个环上有一个蓝色参数点和一个白色动点
4. 玩家移动鼠标到蓝色参数点位置，等待白色动点转到判定弧内
5. 点击 = 激活该参数
6. 全部激活 = 启动跃迁，胜利

### 视觉风格

- 飞船：SVG 六边形图标（星环号），中心 `#7fd4e8`
- 激活环：SVG `<circle>` 描边，无填充
- 蓝色参数点：填充 `#7fd4e8`，光晕脉动
- 白色动点：填充 `#e8e8e8`，沿环旋转
- 判定弧：30° 绿色 `#a8e8c8` 半透明
- 进度条：左上角，宽度 200px，高度 4px，填充 `#d9455f`
- 背景：使用 CSS `transform: scaleY()` 表现二维化压扁效果（从 1 渐变到 0.02）

### 胜利动画

- 星环号跃迁：蓝色光晕扩散
- 二维化太阳系作为背景：所有 SVG 元素 `scaleY` 压扁到接近 0
- 文字逐字浮现："曲率引擎启动……星环号进入光速……二维化的太阳系在身后归于沉寂。"

## 整体设计思路（v2.0 强化版）

### 视角切换

三体视角（第1、2、4关）和地球视角（第3关）交替：
- 三体视角关卡：**冷色调**（`#05060a` 深底 + `#7fd4e8` 冷青）+ 失真音效
- 地球视角关卡：**暖色调**（`#0a0d14` 略深蓝 + `#e8d87f` 暖黄）+ 人声

### 玩法风格

以"反应 + 判断"类玩法为主。

### 剧情呈现

- 大事件通过过场动画或文字交代
- 关键事件做成可玩关卡
- **不复用《三体•低语》已有的"红岸""黑暗森林""二向箔"具体剧情**（避免重复）
- 本游戏聚焦"信号监测员""飞船驾驶员""威慑纪元""多环 QTE"四个差异化体验

### 难度递进

| 关卡 | 玩法 | 难度 | 时长 |
|------|------|------|------|
| 第1关 | 扫雷 10×15 + 声音定位 | ★★ | 3-4 分钟 |
| 第2关 | 飞船潜行 + 引力弹弓 | ★★★ | 3-4 分钟 |
| 第3关 | 真假威胁判断 | ★★★★ | 3-5 分钟 |
| 第4关 | 多环 QTE | ★★★★★ | 3-4 分钟 |

### 重玩价值

- **黑暗森林威慑度总评分**（S/A/B/C）——通关后根据四关表现综合评定
- **ETO 隐藏线索**——每关随机掉落隐藏信息碎片，集齐解锁隐藏剧情
- **多结局分支**——根据玩家选择解锁不同结局文案

### 可访问性

- **色觉障碍模式**：替换色彩编码为形状/纹理编码
- **单声道音频**：支持耳机单声道输出
- **字幕提示**：所有音效有可视反馈兜底
- **难度选择**：三档难度覆盖新手到老玩家

## 验收清单

- [ ] 第1关 10×15 网格可正常点击 + 声音反馈

- [ ] 第4关 3/4/5 环按难度显示
- [ ] 第4关各环异速（差异 ≥ 30%）
